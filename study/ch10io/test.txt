자바 - 데이터의 형태 길이
기본형 - 길이 정해져 있다. - 값
부울형
 * boolean - true / false - 1byte
문자형
 * char - 'A' : 한 문자 - 2byte : 값이 없다.
숫자형
 * 정수 - byte - short - <int> - long : 부호 + 값
 * 실수 - float - <double> : 부호 + 지수 + 값

참조형 - 길이 정해져 있지 않다. - 주소
주소가 비어있다. - null
참조형 변수가 null인데 사용하려고 하면 null point Exception이 발생하게 됨.

작은 데이터 -> 큰 데이터 : 자동 캐스팅이 된다. : 서로 연관이 있어야 한다. : 데이터 유실이 없는 경우.
큰 데이터 -> 작은 데이터 : 강제 캐스팅을 해야 함. : 서로 연관이 있어야 한다. : 데이터 유실이 있음. : (캐스팅 타입) 변수

상수 변수
final - 클래스, 메서드, 변수
final class - 구성원 변경 불가
final method(){} - 처리문 변경 불가
final 변수 - 값 변경 불가
/** */ - 범위 주석 - 자동 문서화 작업 가능
/* */ - 범위 주석 - 개발자만 보기 위해 - 자동 문서화 불가
// 한 줄 주석

10진법 (0~9) - 사람
2진법 (0~1) - 컴퓨터 내부
-> 보기가 힘들어 3자리씩 묶은 것이 8진법 (0~7) 000 ~ 111
-> 4자리씩 묶은 것이 16진법 (0~15) 0000 ~ 1111
-> 사람이 보기 편하기 위해서 하는 것.

양수인 경우 신경 안써도 됨.
음수인 경우 -> 부호 + 값
0000 -> +000 
0010 -> +010 -> +2

     1101 - 1의 보수는 1이면 0, 0이 서로 보전해서 완전체인 1로 만드는 것 0->1, 1->0
